# AI Self-Review by Spec

## Automated Review

Letting AI self-review based on specifications is an efficient practice that can be done before manual review.

## Operation Method

```markdown
## Specification
[Your specification]

## Code
[AI-generated code]

## Task
Please verify whether the code meets the following specification requirements:

1. [Specification requirement 1] - Met?
2. [Specification requirement 2] - Met?
3. ...

## Output
- Check result for each specification item
- Problems found
- Fix suggestions
```
