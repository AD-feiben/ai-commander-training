# Give Specs to AI

## Core SDD Practice

In SDD methodology, the most important practice is to give the complete specification to AI and let AI generate code based on the specification.

## Operation Steps

### 1. Prepare Specification

Ensure the specification is complete, clear, and verifiable.

### 2. Write Prompt

```markdown
## Task
Generate code based on the following specification.

## Specification
[Your complete specification]

## Requirements
1. Strictly follow the specification
2. Add necessary error handling
3. Code should be complete and runnable
4. Include necessary type definitions
```

### 3. Verify Results

Check if the generated code meets the specification.
