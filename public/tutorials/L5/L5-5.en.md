# AI-Assisted Debugging

## AI is a Good Debugging Helper

AI can not only generate code but also help you locate and fix problems.

## Debug Prompt

```markdown
## Problem Description
[Describe problem]

## Error Message
[Error stack]

## Related Code
[Related code]

## Task
1. Analyze possible causes
2. Provide fix solutions
3. Explain principles
```

## Debug Process

```
Discover error → Describe to AI → AI analyzes → Fix → Verify
```
