# 什么是 AI Skills

## Skills vs 通用助手

**Skill**（技能）是一个定义好的 AI 角色和能力，而通用助手则是「什么都能做但什么都不专」。

2025 年 12 月，Agent Skills 已成为开放规范，主流工具都已支持。

## 为什么需要 Skills？

| 通用助手 | Skills |
|---------|--------|
| 什么都能做 | 专注做好一件事 |
| 每次都要说明背景 | 一次配置，长期使用 |
| 质量不稳定 | 输出更一致 |

## 常见 Skill 类型

### 1. 代码审查 Skill

```
角色：专业代码审查员
专长：安全、性能、代码规范
输出：结构化的审查报告
```

### 2. 文档写作 Skill

```
角色：技术文档专家
专长：API 文档、README
输出：清晰的文档
```

### 3. 测试工程师 Skill

```
角色：测试工程师
专长：单元测试、集成测试
输出：完整的测试用例
```
