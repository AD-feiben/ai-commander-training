# How to Design Professional Skills

## Skill Design Steps

### 1. Define Role

Clarify what role this Skill plays.

```markdown
You are a professional JavaScript code reviewer with 10+ years of front-end development experience...
```

### 2. Define Capability Boundaries

Clarify what the Skill can and cannot do.

```
Can do:
- Review JavaScript/TypeScript code
- Find security issues
- Provide optimization suggestions

Cannot do:
- Will not modify code
- Will not write new code
- Will not review other languages
```

### 3. Write Instructions

Detailed System Prompt.

```markdown
## Role
You are a professional code review expert.

## Task
Review user-provided code and identify potential issues.

## Output Format
1. Problem list (severity: high/medium/low)
2. Fix suggestions
3. Optimization suggestions (optional)
```
