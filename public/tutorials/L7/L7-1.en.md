# What are AI Skills

## Skills vs General Assistant

**Skill** is a defined AI role and capability, while a general assistant is "can do everything but specializes in nothing".

In December 2025, Agent Skills became an open standard, supported by mainstream tools.

## Why Do We Need Skills?

| General Assistant | Skills |
|-------------------|--------|
| Can do everything | Focus on doing one thing well |
| Need to explain context every time | Configure once, use long-term |
| Unstable quality | More consistent output |

## Common Skill Types

### 1. Code Review Skill

```
Role: Professional code reviewer
Expertise: Security, performance, code standards
Output: Structured review report
```

### 2. Documentation Writing Skill

```
Role: Technical documentation expert
Expertise: API docs, README
Output: Clear documentation
```

### 3. Test Engineer Skill

```
Role: Test engineer
Expertise: Unit tests, integration tests
Output: Complete test cases
```
