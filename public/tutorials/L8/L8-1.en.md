# What is Sub-agent

## Commander and Executor

**Sub-agent** is an important concept in AI collaboration systems. The main Agent acts as the "commander," while Sub-agents act as "executors."

## Sub-agent vs Skills

| | Skills | Sub-agent |
|---|--------|-----------|
| Independence | Low (reuses main Agent) | High (works independently) |
| Complexity | Simple tasks | Complex tasks |
| Communication | One-way | Two-way |
