# 什么是 Sub-agent

## 指挥官与执行者

**Sub-agent**（子代理）是 AI 协作系统中的重要概念。主 Agent 充当「指挥官」，Sub-agent 充当「执行者」。

## Sub-agent vs Skills

| | Skills | Sub-agent |
|---|--------|-----------|
| 独立性 | 低（复用主 Agent） | 高（独立工作） |
| 复杂度 | 简单任务 | 复杂任务 |
| 通信 | 单向 | 双向 |
