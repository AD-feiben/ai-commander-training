# Practice: Build Code Review Team

## Exercise Objective

This section will guide you through designing a multi-Sub-agent code review team.

## Team Design

### Master Agent: Architect

```
Responsibilities:
- Receive code
- Distribute to Sub-agents
- Integrate review results
- Generate final report
```

### Sub-agent 1: Security Reviewer

```
Expertise: Security vulnerability detection
- SQL injection
- XSS
- Permission issues
```

### Sub-agent 2: Performance Reviewer

```
Expertise: Performance optimization
- N+1 queries
- Memory leaks
- Algorithm complexity
```
