# 实战：构建代码审查团队

## 练习目标

本节将带你设计一个多 Sub-agent 的代码审查团队。

## 团队设计

### 主 Agent：架构师

```
职责：
- 接收代码
- 分发给 Sub-agent
- 整合审查结果
- 生成最终报告
```

### Sub-agent 1：安全审查员

```
专长：安全漏洞检测
- SQL 注入
- XSS
- 权限问题
```

### Sub-agent 2：性能审查员

```
专长：性能优化
- N+1 查询
- 内存泄漏
- 算法复杂度
```
