# Designing Sub-agent Systems

## Master-Slave Agent Architecture

Designing Sub-agent systems requires clarifying master-slave relationships.

### 1. Define Master Agent

```
Master Agent: Project Leader
- Understand overall requirements
- Decompose tasks
- Assign to Sub-agents
- Integrate results
```

### 2. Define Sub-agent

```
Sub-agent 1: Front-end Development
- Only responsible for front-end code
- Follow front-end standards

Sub-agent 2: Back-end Development
- Only responsible for back-end code
- Follow back-end standards
```
