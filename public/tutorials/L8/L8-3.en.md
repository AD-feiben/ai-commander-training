# Decomposing Complex Tasks into Subtasks

## Task Tree Design

Complex tasks need to be decomposed into tree-like structures.

## Decomposition Method

### 1. Identify Main Steps

```
Build E-commerce Website
  ├── User Module
  ├── Product Module
  ├── Order Module
  └── Payment Module
```

### 2. Identify Dependencies

```
User Module ← Product Module ← Order Module ← Payment Module
  
Dependency order: User → Product → Order → Payment
```
