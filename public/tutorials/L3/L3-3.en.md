# AI-Assisted Requirements Analysis

## AI is a Good Helper for Requirements Analysis

You can not only ask AI technical questions but also let it help you with requirements analysis.

## Requirements Analysis Prompt

```markdown
Please help me analyze the following requirements and identify functional points and technical points:

## Requirement Description
[Your requirement]

## Please Output
1. Feature list (grouped by module)
2. Data model (if needed)
3. Technical suggestions
4. Potential risks
5. Missing issues (please list points you don't have clear information about)
```

## Example

```
Requirement: Build a blog system
```

Let AI analyze:

```markdown
## Feature Modules

### User Module
- Registration/Login
- Profile management

### Article Module
- Create article
- Edit article
- Delete article
- Article list
- Article details
- Categories/Tags

### Comment Module
- Post comment
- Reply to comment

## Data Model

- User: id, username, email, password, avatar
- Post: id, title, content, author_id, created_at
- Comment: id, content, post_id, user_id, parent_id

## Technical Suggestions

- Frontend: React + Markdown editor
- Backend: RESTful API
- Database: PostgreSQL
- Images: Use object storage

## Potential Risks

- Performance: Large number of articles requires pagination
- Security: XSS attacks need protection
- SEO: Requires server-side rendering or static generation
```

## Human-AI Collaboration

```
You: Provide requirements
AI: Analyze features, identify issues
You: Confirm and supplement
AI: Analyze again
...Iterate...
```
