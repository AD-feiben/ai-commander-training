# Multi-AI Multi-Module Collaboration

## Parallel and Serial

Multiple modules can be developed in parallel or serial.

### Parallel

```
AI 1 → User Module
AI 2 → Product Module  
AI 3 → Order Module

Develop simultaneously, save time
```

### Serial

```
Base Module → Business Module → Advanced Module

Have dependencies, need to follow order
```
