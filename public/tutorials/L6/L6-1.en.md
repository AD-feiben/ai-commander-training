# Spec Architecture for Large Projects

## Layered Specifications

Large projects require layered specifications, from overall to local.

## Specification Levels

### Level 1: Project Specification

```markdown
# E-commerce Website Project Specification

## Project Overview
[Project goals]

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js
- Database: PostgreSQL

## Module Division
- auth: Authentication module
- user: User module
- product: Product module
- order: Order module
```

### Level 2: Module Specification

```markdown
# Product Module Specification

## Feature List
1. Product list (pagination, filtering)
2. Product details
3. Product search
4. Product categories
5. Product management (CRUD)

## Data Model
Product: id, name, description, price, category_id, stock, images
```
