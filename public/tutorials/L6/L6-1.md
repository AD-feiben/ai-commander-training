# 大型项目的规范架构

## 分层规范

大型项目需要分层规范，从整体到局部。

## 规范层次

### 1. 项目级规范

```markdown
# 项目规范

## 项目概述
[项目目标]

## 技术栈
- 前端：React + TypeScript
- 后端：Node.js + Express
- 数据库：PostgreSQL

## 模块划分
- auth：认证模块
- user：用户模块
- product：商品模块
- order：订单模块
```

### 2. 模块级规范

```markdown
# 用户模块规范

## 功能列表
- 用户注册
- 用户登录
- 用户信息管理

## API 设计
- POST /api/users/register
- POST /api/users/login
- GET /api/users/me

## 数据模型
User: id, username, email, password_hash, created_at
```
