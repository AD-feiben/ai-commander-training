# Module Division & Dependencies

## Modular Design

Large projects require good modular design.

## Module Division Principles

1. **Single Responsibility** - Each module does only one thing
2. **High Cohesion Low Coupling** - Tightly related internally, loosely dependent externally
3. **Clear Dependencies** - Clear dependency relationships between modules
