# 提示词模板库

## 常用提示词模板

这里收集了一些常用的提示词模板，可以直接使用或根据需要修改。

## 1. 代码生成模板

```markdown
## 角色
你是一个 [语言] 专家。

## 任务
写一个 [具体功能] 函数/类。

## 要求
- 使用 [语言/框架]
- 满足以下约束：[具体约束]
- 遵循 [代码规范]

## 输入/上下文
[如果有的话]

## 输出
[期望的格式]
```

## 2. 代码审查模板

```markdown
## 角色
你是一个资深代码审查专家。

## 代码
[代码]

## 审查要求
1. 找出潜在 bug
2. 发现安全问题
3. 识别性能问题
4. 检查代码规范

## 输出格式
- 问题列表
- 严重程度
- 修复建议
```

## 3. 解释代码模板

```markdown
## 代码
[代码]

## 任务
详细解释这段代码：
1. 功能
2. 关键逻辑
3. 可能的优化点

## 目标读者
[初学者/有经验的开发者]
```

## 4. Debug 模板

```markdown
## 问题描述
[描述问题]

## 错误信息
[错误信息]

## 相关代码
[代码]

## 任务
1. 分析可能原因
2. 提供解决方案
3. 解释原理
```
