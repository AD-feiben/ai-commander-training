# Few-shot 示例技巧

## 什么是 Few-shot？

Few-shot（一少样本学习）是指在提示词中给出几个示例，帮助 AI 理解你期望的输出格式或行为。

## 为什么有效？

AI 通过示例可以更好地理解：
- 输出格式
- 风格要求
- 具体要求

## 使用场景

### 1. 指定输出格式

```
任务：将用户信息转换为 JSON

示例：

输入：张三，男，1990年1月1日生
输出：{"name": "张三", "gender": "男", "birthday": "1990-01-01"}

输入：李四，女，1995年5月20日生
输出：
```

这样 AI 就会按照同样的格式输出。

### 2. 指定代码风格

```
任务：写一个 React 组件

示例：

// 示例组件
const Button = ({ label, onClick }) => (
  <button onClick={onClick} className="btn">
    {label}
  </button>
);

// 你的组件：写一个 Input 组件
```

### 3. 指定回复风格

```
任务：用幽默的方式解释技术概念

示例：

用户：什么是 JavaScript？
AI：JavaScript 就像一个会变魔术的厨师，能让静态的网页变得生动有趣。

用户：什么是 React？
AI：
```

## 使用技巧

1. **示例要相关** - 使用与任务相似的示例
2. **示例要正确** - 确保示例是正确的
3. **示例数量** - 通常 2-5 个足够
4. **格式一致** - 所有示例格式要统一
