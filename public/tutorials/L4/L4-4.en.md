# Iterative Prompting

## What is Iterative Prompting

Iterative prompting refers to gradually optimizing AI's output through multi-turn dialogue. Give AI feedback each round to let it improve.

## Why Iteration is Needed

AI's first generated content may not be perfect. Through iteration, you can:
- Gradually improve
- Correct errors
- Optimize details

## Iteration Pattern

```
Round 1: Generate first draft
  You → AI: Help me write a function
  AI → You: First draft code

Round 2: Point out issues
  You → AI: First round has a bug, edge cases not handled
  AI → You: Corrected code

Round 3: Further optimization
  You → AI: Can performance be improved?
  AI → You: Optimized code
```

## Iteration Tips

1. **Specific feedback** - Point out specific issues, don't just say "bad"
2. **Change one thing at a time** - Focus on one improvement point per iteration
3. **Be clear about what you want** - Tell AI what you want, not what you don't want
