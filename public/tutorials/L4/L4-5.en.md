# Prompt Template Library

## Common Prompt Templates

Here are some commonly used prompt templates that you can use directly or modify as needed.

## 1. Code Generation Template

```markdown
## Role
You are a [language] expert.

## Task
Write a [specific function] function/class.

## Requirements
- Use [language/framework]
- Meet the following constraints: [specific constraints]
- Follow [code standards]

## Input/Context
[If any]

## Output
[Expected format]
```

## 2. Code Review Template

```markdown
## Role
You are a senior code review expert.

## Code
[Code]

## Review Requirements
1. Find potential bugs
2. Discover security issues
3. Identify performance problems
4. Check code standards

## Output Format
- Problem list
- Severity level
- Fix suggestions
```

## 3. Explain Code Template

```markdown
## Code
[Code]

## Task
Explain this code in detail:
1. Function
2. Key logic
3. Possible optimization points

## Target Audience
[Beginners/Experienced developers]
```

## 4. Debug Template

```markdown
## Problem Description
[Describe problem]

## Error Message
[Error message]

## Related Code
[Code]

## Task
1. Analyze possible causes
2. Provide solutions
3. Explain principles
```
