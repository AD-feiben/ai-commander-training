# Structured Prompts

## What are Structured Prompts

Structured prompts are a way of organizing instructions that help AI understand your needs more accurately.

## Core Elements: RCTCs

Structured prompts contain four core elements:

| Element | English | Description |
|---------|---------|-------------|
| Role | Role | What role AI plays |
| Context | Context | Relevant background information |
| Task | Task | What task to complete |
| Constraint | Constraint | What limitations exist |

## Example

### ❌ Bad

```
Write a function to validate passwords
```

### ✅ Good

```
You are a professional security engineer (Role)

Need to write a password validation function for a user system (Context)

Please write a password validation function in TypeScript that meets the following requirements (Task):
1. At least 8 characters long
2. Contains uppercase letters
3. Contains lowercase letters
4. Contains numbers
5. Contains special characters

Please return boolean, true means validation passed (Constraint)
```

## Template

```markdown
## Role
[You are a...]

## Context
[Relevant context...]

## Task
[What to complete...]

## Constraints
1. ...
2. ...
3. ...
```

## Practical Application

```markdown
## Role
You are a senior front-end architect, specializing in React performance optimization.

## Context
We are developing a large SaaS application with 100+ pages.

## Task
Please help me review the following React component code and identify performance issues:

[Code...]

## Constraints
1. Focus on rendering performance
2. Identify memory leak risks
3. Provide specific optimization suggestions
```
