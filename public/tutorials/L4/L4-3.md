# 思维链 CoT

## 什么是思维链？

思维链（Chain of Thought，CoT）是一种提示技术，通过让 AI 先展示推理过程，再给出答案，提高回答质量。

## 核心思想

不要只给答案，要展示思考过程。

## 简单对比

### ❌ 没有 CoT

```
用户：5 * (3 + 2) - 4 = ?
AI：11
```

### ✅ 有 CoT

```
用户：5 * (3 + 2) - 4 = ?
请先展示计算步骤，再给出答案。

AI：
1. 先算括号内：3 + 2 = 5
2. 再算乘法：5 * 5 = 25
3. 最后算减法：25 - 4 = 21

答案：21
```

## 适用场景

### 1. 复杂推理

```
请计算：从 1 加到 100 的和。请展示推理过程。
```

### 2. 代码审查

```
请审查以下代码，先分析可能的问题，再给出结论。

代码：[...]
```

### 3. 问题诊断

```
我的服务器响应很慢，请先分析可能原因，再给出排查步骤。
```

## 如何使用

在提示词中添加：

```markdown
请先分析问题，展示你的思考过程，然后给出答案或建议。
```

或者：

```markdown
请按以下步骤思考：
1. ...
2. ...
3. ...
最后给出结论
```

## 效果

| 场景 | 无 CoT | 有 CoT |
|------|--------|--------|
| 数学问题 | 容易出错 | 更准确 |
| 代码审查 | 可能遗漏 | 更全面 |
| 问题诊断 | 治标不治本 | 找到根因 |
