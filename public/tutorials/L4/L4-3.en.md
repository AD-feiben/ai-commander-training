# Chain of Thought (CoT)

## What is Chain of Thought?

Chain of Thought (CoT) is a prompting technique where AI shows its reasoning process before giving the answer, improving response quality.

## Core Idea

Don't just give the answer, show the thinking process.

## Simple Comparison

### ❌ Without CoT

```
User: 5 * (3 + 2) - 4 = ?
AI: 11
```

### ✅ With CoT

```
User: 5 * (3 + 2) - 4 = ?
Please show calculation steps before giving the answer.

AI:
1. First calculate parentheses: 3 + 2 = 5
2. Then multiplication: 5 * 5 = 25
3. Finally subtraction: 25 - 4 = 21

Answer: 21
```

## Applicable Scenarios

### 1. Complex Reasoning

```
Please calculate: Sum from 1 to 100. Show your reasoning process.
```

### 2. Code Review

```
Please review the following code, analyze potential issues first, then give conclusions.

Code: [...]
```

### 3. Problem Diagnosis

```
My server response is very slow. Please analyze possible causes first, then provide troubleshooting steps.
```

## How to Use

Add to your prompt:

```markdown
Please analyze the problem first, show your thinking process, then give the answer or suggestion.
```

Or:

```markdown
Please think in the following steps:
1. ...
2. ...
3. ...
Finally give the conclusion
```

## Effect

| Scenario | Without CoT | With CoT |
|----------|-------------|----------|
| Math problems | Prone to errors | More accurate |
| Code review | May miss issues | More comprehensive |
| Problem diagnosis | Treat symptoms only | Find root cause |
