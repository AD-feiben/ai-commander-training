# Few-shot Example Techniques

## What is Few-shot?

Few-shot (few-shot learning) refers to giving AI a few examples in the prompt to help it understand the desired output format or behavior.

## Why It Works

AI can better understand through examples:
- Output format
- Style requirements
- Specific requirements

## Usage Scenarios

### 1. Specifying Output Format

```
Task: Convert user information to JSON

Example:

Input: Zhang San, Male, Born January 1, 1990
Output: {"name": "Zhang San", "gender": "Male", "birthday": "1990-01-01"}

Input: Li Si, Female, Born May 20, 1995
Output:
```

This way AI will output in the same format.

### 2. Specifying Code Style

```
Task: Write a React component

Example:

// Example component
const Button = ({ label, onClick }) => (
  <button onClick={onClick} className="btn">
    {label}
  </button>
);

// Your component: Write an Input component
```

### 3. Specifying Response Style

```
Task: Explain technical concepts in a humorous way

Example:

User: What is JavaScript?
AI: JavaScript is like a magician chef who can make static web pages come alive.

User: What is React?
AI:
```

## Usage Tips

1. **Examples should be relevant** - Use examples similar to the task
2. **Examples should be correct** - Ensure examples are correct
3. **Number of examples** - Usually 2-5 is enough
4. **Consistent format** - All examples should have uniform format
