## 2025 New Trend: Vibe Coding

> Vibe Coding = Philosophy + Agent (Tools) + MCP (Protocol)

Vibe Coding is a brand new programming philosophy:
- Describe requirements in natural language
- AI Agent autonomously completes development
- Humans are responsible for review and decision-making

## Selection Recommendations

```
Light usage → GitHub Copilot
Deep development → Claude Code / Cursor
Complex projects → Windsurf
Need external integration → MCP
```
