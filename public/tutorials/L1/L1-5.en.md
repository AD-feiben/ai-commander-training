# Write Your First AI Prompt

## Start Practicing

It's time to write your first AI prompt! This is a simple exercise to help you understand how to communicate effectively with AI.

## A Good Prompt

```markdown
Write a TypeScript function for validating email addresses.

Requirements:
1. Use regular expression validation
2. Return boolean
3. Function name should be isValidEmail
4. Add JSDoc comments
```

This prompt includes:
- **Task**: What to do (validate email)
- **Constraints**: What requirements (TypeScript, regex, return boolean)
- **Format**: Specific format (function name, comment requirements)

## A Bad Prompt

```markdown
Write an email validation function
```

This prompt is too vague. AI doesn't know:
- What language to use
- What format to return
- What specific requirements there are

## Try It Now

Now you can try:
1. Open Claude Code, Cursor, or other AI tools
2. Enter the "good prompt" above
3. Observe AI's output

## Summary

A good AI prompt contains three elements:

| Element | Description | Example |
|---------|-------------|---------|
| Task | What to do | Validate email |
| Constraints | What requirements | TypeScript, regex |
| Format | Specific format | Return boolean, add JSDoc |
