## Overview

AI Commander Training Camp is a free and open-source AI programming skills learning platform that helps developers transition from traditional coders to AI Commanders who can effectively direct AI tools to complete complex programming tasks.

### Core Skills

- **Spec-Driven Development (SDD)**: Write high-quality technical specifications to guide AI in generating code that meets expectations
- **Prompt Engineering**: Master core techniques for efficient communication with AI
- **Multi-Agent Collaboration**: Design and manage collaborative systems with multiple AI agents

## Live Demo

- **Website**: https://ai-commander-training.netlify.app/
- **Course**: 9 chapters, 45 lessons

## Quick Start

### Prerequisites

- Node.js >= 18
- npm >= 9

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Tech Stack

- **Framework**: React 19 + React Router 7
- **Language**: TypeScript 5.9
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS 3.4
- **State Management**: Zustand 5
- **Markdown Rendering**: react-markdown + remark-gfm
- **Code Highlighting**: react-syntax-highlighter
- **Diagrams**: Mermaid
- **Comments**: Giscus
- **Deployment**: Netlify / Vercel
